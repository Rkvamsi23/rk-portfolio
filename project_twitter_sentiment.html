Watching Twitter Turn During Events

Python NLP Power BI

Goal

Build a small pipeline to see how sentiment and topics on Twitter (now X) shift during a live event, and how quickly the crowd mood can turn.

Pipeline

  • Streamed tweets with selected keywords and hashtags using the Twitter API
  • Cleaned and normalised the text
  • Applied basic sentiment analysis and simple topic grouping
  • Stored results with timestamps for later analysis
  • Built a Power BI dashboard to visualise sentiment and topic share over time

Interesting patterns

  • Sharp sentiment swings around key announcements or controversial moments
  • Topic clusters forming and dissolving as memes spread
  • Small but loud negative clusters that shaped the feeling of the feed out of proportion to their size

Skills used

  • Python for data collection and processing
  • Simple NLP, tokenisation and sentiment scoring
  • Power BI for interactive visualisation

Why it matters for my current work

This was my first experience watching text as a moving crowd in real time. Later, when I started thinking about stand up clips and platform algorithms, this project gave me a mental model for how public mood can flip fast and how a single joke can become a flashpoint.
